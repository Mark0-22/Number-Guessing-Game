# Number Guessing Game
 Number Guessing Game  , runs in CLI

Clone repository:

    git clone https://github.com/Mark0-22/Number-Guessing-Game.git
    cd Number-Guessing-Game

Install dependencies:

    npm install

Run this command in CLI to start the game:

    node start
